# CSS Advanced Project

This project complements a provided HTML document with a custom CSS stylesheet.  
The goal is to apply responsive design, clean layout, and consistent styling based on the given Figma design.

---

## Objectives
- Style the header, banner, quotes, videos list, membership, FAQ, and footer sections.
- Ensure responsive design using media queries for mobile and desktop.
- Keep CSS simple, clean, and W3C‑compliant.
- Match the design specifications provided in the Figma file.

---

## Project Structure
- `index.html` → Provided HTML structure (base document).
- `styles.css` → Custom stylesheet written for this project.
- `README.md` → Documentation for the project.

## Requirements
- All files must end with a new line.
- No external libraries (Bootstrap, React, Vue, etc.) — only pure HTML/CSS/JavaScript.
- Code must validate with the W3C Validator.
- Fonts: **Source Sans Pro** and **Spin Cycle OT** (as specified in the design).
- Values may be rounded if necessary.

## Learning Outcomes
By completing this project, you should be able to:
- Explain what CSS is and how to add style to an element.
- Use classes and selectors effectively.
- Compute CSS specificity values.
- Apply box properties in CSS.
- Understand how browsers load and render webpages.
- Build responsive layouts with flexbox and media queries.
